== Description ==

Adicione e opção CANCELAR PEDIDO(S) na listagem de pedidos de seu Woocommerce e facilite a gestão de sua Loja.

== Installation ==

1. Faça o upload do plugin através do painel do WordPress, indo em "Plugins" > "Adicionar Novo" ou Instale pelo Repositório WordPress
2. Ative o plugin indo até o menu "Plugins";

Pronto, a opção ja aparecerá na sua lista de ações em massa.

== Requirements: ==

- [PHP 7.2+](https://www.php.net/releases/7_2_0.php) é a versao MINIMA requerida para este plugin).
